Personal Projects:

• Rubik's Cube solver:

One project which I decided to take on, was to write a program, which can see a camera input, interpret how the Rubik's cube is structured and give you a clear sequence of steps which need to be followed to solve the cube. The program cube_image.py takes care of the image processing while the programm rubix_cube_loesen.py takes care of the steps to solve the cube and the visual output window.

• Binomial Distribution

I programmed this program which calculates the binomial distribution for my Math class, and then showed it to my math teacher, he liked it very much. mathe_binominalverteilung.py

• Class Enterer program

During the time of online classes, I wanted to practice my skills with useful projects, so I decided to code a program to automatically join me into every days videoconferences. get_into_class.py

• Tetris Game

Learning how to use the Pygame python module, I practiced on some basic projects, amongst them was a game of Tetris. tetris.py

• Games Talente:

There is a private German organization called "Games Talente" which organizes a contest yearly, where people can submit video games, and the best ones, get chosen for a workshop going on for a week. I submited the following video game (in German) written in Python code with Pygame, and they accepted me. It was a great week and I learned a lot. Games_Talente_Projekt.py
